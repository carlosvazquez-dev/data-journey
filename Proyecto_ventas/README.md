# Proyecto de análisis de ventas

## Objetivo
Analizar datos de ventas para identificar el producto con mayor ingreso, la ciudad con mayores ventas y el precio promedio por producto.

## Herramientas
- Python
- pandas
- matplotlib

## Archivos
- dia10_proyecto.py
- dia10_conclusiones.txt
- ventas_reales.csv

## Hallazgos
- Producto con mayor ingreso: Filtro
- Ciudad con mayores ventas: Torreon
- Producto con mayor precio promedio: Bateria

## Conclusión
Este proyecto muestra un análisis básico de ventas usando limpieza, agrupación y visualización de datos.
